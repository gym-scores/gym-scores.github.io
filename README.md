
# gym-scores

For a few years, kmkeirns and I have been compiling elite gymnastic scores, since no official repository exists and it can be difficult to find complete data on some meets. We have each been doing this inside Google Sheets, but as you can imagine, a browser spreadsheet is not the most viewer-friendly experience, and the spreadsheet itself quickly becomes cumbersome as data piles up.

This project is meant to address the ease-of-use conundrum. For now, it displays data as kmkeirns' spreadsheet does: a giant table of every score this year on a given apparatus, with pagination so you're not scrolling through all the thousands of scores in the window. These tables can be sorted by any column (defaulting to total score), and you can use the search bar as a filter. Bear in mind that the search function right now is fairly primitive. Searching for "ITA" will bring up every Italian score, but it will also bring up Vitaliy Guimaraes because his name includes the letters "ita." Searching for a meet name when it was multi-day doesn't always exclude all other days of that meet either. Not sure what's up with that yet, but it will at least give you fewer results to sift through.

In the future, I hope to add some more features, like meet results, FIG-only data, and an athlete database. I'm learning as I go, so please be patient.
